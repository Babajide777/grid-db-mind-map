## Run Locally

Clone the project

```bash
https://github.com/Babajide777/grid-db-mind-map.git
```

Go to the project directory

```bash
 cd client
```

Install dependencies

```bash
 npm install
```

Start the server

```bash
  npm start
```

## Tech Stack

frontend: React, React-icon, Reactflow, Material-UI, RTK Query, Uniqid

Backend: ExpressJS, Joi

Database: Grid DB
