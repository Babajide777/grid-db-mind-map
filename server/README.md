## Run Locally

Clone the project

```bash
  git clone https://github.com/Babajide777/grid-db-mind-map.git
```

Go to the project directory

```bash
 cd server
```

Build docker compose file

```bash
 docker-compose build
```

Run docker

```bash
 docker-compose up
```
